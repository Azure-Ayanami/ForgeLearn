* 常态下基础能力降低
* 成瘾性

## 新增功能

* 夺取生物能力。先喂动物吃 *巨花魔芋* ，然后使用 *佛焰苞* 精粹业力，转化为能力

* 转化过程分为 X 个阶段，以下是 ==【通用模型 - 羊】== 的阶段
